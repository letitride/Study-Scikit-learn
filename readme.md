# 回帰

## 線形回帰

```LinearRegression```

重回帰も同じモデルとなる。

## リッジ回帰

線形回帰のオーバーフィットに対して正則化するモデル

```Ridge```

alpha値で正則化の強度を変更できる

## ロッソ回帰

特徴量を削減して正則化するモデル

```Lasso```

# 分類

## ロジスティック回帰

回帰とつくが分類アルゴリズム

```LogisticRegression```

C値で正則化の強度を変更できる

## サポートベクタマシン

```LinearSVC```

## 決定木

```DecisionTreeClassifier```
